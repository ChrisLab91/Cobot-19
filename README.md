#### Ubuntu installation
- python3.6
- `pip install SpeechRecognition`
- `apt-get install portaudio19-dev && sudo pip install pyaudio`
- `pip install pyaudio`
- `pip install git+https://github.com/nateshmbhat/pyttsx3`
